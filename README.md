# 内核设计赛: Lost Wakeup OS

## 项目构建

- 在项目根目录下通过以下命令构建项目

    ```bash
    make all
    ```
- 如果添加/修改了用户程序，可以通过以下命令重新生成文件系统镜像

    ```bash
    make image
    ```

- 若没有修改用户程序，仅对内核代码进行了改动，需要单独编译内核，使用以下两条命令之一

    ```bash
    make kernel
    make
    ```
    此命令可以确保文件系统的持久化，不会导致先前对文件系统镜像的修改被覆盖


- 使用
  ```bash
  make clean-all
  ```
    清除所有的内核、用户编译结果以及文件系统镜像
